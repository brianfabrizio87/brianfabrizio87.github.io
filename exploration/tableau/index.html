Tableau API

Exploring the Tableau API

Overview

Tableau is powerful business intelligence software focused on visual data exploration.
This page explores some basic API methods focused primarily on embedding visualizations in HTML pages supporting the creation of information portals and dashboards. jQuery and Bootstrap provide functionality external to embedded visualizations.

For information regarding the development of visualization types, please refer to the extensive documentation on the official Tableau Training site.

Views

A view is simply a pre-existing, embedded Tableau Sheet, Dashboard, or Story stored on a Tableau server.
A view, in the sense that I am referring, is a visualization stored within a Tableau sheet. When sheets are embedded with inherent tabs, the tabs can be hidden through options, and each tab (sheet) visualization can be shown or hidden based on custom UI components.

    Notice that you can interact with the embedded visualization with controls provided by Tableau without any additional coding.

Filtering

Similar to views, filters can be programmatically created and tied to UI components. Sheet names are passed to your functions as parameters, with the function tied to the UI component that launches (calls) the specific function.

  • Single Dimension Filter
  • By passing a dimension category to a custom function, it is possible to filter a visualization based on user-defined parameters. In this case, we filtered on a single dimension.
  • Multiple Dimensions Filter
  • By passing a dimension category to a custom function, it is possible to filter a visualization based on user-defined parameters. In this case, we filtered on multiple dimensions.
  • Marking Filter
  • Marking is similar to filtering in that the desired dimension is identified, however the dimension is highlighted with all other dimensions still visible with sublte opacity set, making the marked dimension stand out visually.
